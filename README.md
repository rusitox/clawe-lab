# Clawe Lab

Repositorio público para proyectos construidos por un equipo de agentes (Software Engineer Lead + sub-agentes).

## Reglas

- **Los requerimientos del software vienen de Rusitox**. No se implementan features no solicitadas.
- Todo cambio entra vía **Pull Request** (PR) y revisión antes de merge a `main`.

## Estructura

- `.github/` — templates y workflows
- `docs/` — ADRs, decisiones, guías
- `projects/` — proyectos (cada uno con su README)
