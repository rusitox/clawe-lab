allowed-tools: Read, Bash, Grep, Glob
description: Root Cause Analysis for debugging issues
---

Investigate: $ARGUMENTS

1. **Reproduce**: Understand the symptoms and reproduction steps
2. **Evidence**: Search logs, code, git history for clues
3. **Trace**: Follow the execution flow to isolate the failure point
4. **Root cause**: Identify the actual cause (not just the symptom)
5. **Fix**: Propose a fix with explanation
6. **Prevention**: Suggest how to prevent recurrence (test, guard, monitoring)
