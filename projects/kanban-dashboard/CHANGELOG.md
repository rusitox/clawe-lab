# Changelog — Clawe Kanban Dashboard

## v0.1.0 — First tagged release

- Kanban web UI (Backlog / To‑do / In progress / Done)
- Drag & drop + CRUD de tarjetas
- Persistencia server-side (API + `tasks.json`) para multi-dispositivo
- Auto-refresh (poll) para reflejar cambios sin refrescar
- Activity feed (API + `activity.json`) para monitorear acciones del equipo
- Filtros globales por **Equipo** y **Agente** (aplican a feed + kanban)
- Colores por equipo configurables (API + `teams.json` + pickers)
- Repo bootstrap + reglas de requerimientos por Issues/PRs
