Login throws 500 on Safari 17

Description

When users attempt to log in using Safari version 17.x, the authentication endpoint returns a 500 Internal Server Error. Chrome and Firefox are unaffected.

Reproduction Steps

  1. Open Safari 17
  2. Navigate to /login
  3. Submit form with valid credentials
