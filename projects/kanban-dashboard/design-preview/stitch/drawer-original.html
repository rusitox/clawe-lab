Login throws 500 on Safari 17

Description

When users attempt to log in using Safari version 17.x, the authentication endpoint returns a 500 Internal Server Error. Other browsers (Chrome, Firefox) are unaffected. Initial logs suggest a failure in processing the User-Agent header or a cookie serialization issue specific to this browser engine.

Repro

  1. Open Safari 17
  2. Navigate to /login
  3. Submit login form with valid credentials
